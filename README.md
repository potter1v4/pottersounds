# 🎵 Potter Sounds - Interface de Autenticação

[cite_start]Uma interface de login moderna e imersiva inspirada em grandes plataformas de streaming, desenvolvida para praticar conceitos avançados de UI Design e manipulação de formulários com JavaScript[cite: 17, 18].

## 🚀 Tecnologias Utilizadas
* [cite_start]**HTML5** e **CSS3** (Variáveis CSS para temas) [cite: 10]
* [cite_start]**Glassmorphism:** Aplicação de efeitos de transparência e desfoque no fundo (backdrop-filter) 
* [cite_start]**JavaScript Puro:** Lógica de validação e interação sem dependências externas [cite: 20]

## 🎨 Diferenciais do Projeto
- [cite_start]**Design Glassmorphism:** Interface elegante com efeito de "vidro fosco".
- [cite_start]**Validação de Formulário:** Verificação em tempo real de campos de e-mail e senha utilizando JavaScript[cite: 20].
- [cite_start]**Responsividade:** Totalmente adaptável para dispositivos móveis e desktop[cite: 12].
- **Acessibilidade:** Foco em contraste e leitura, mesmo com efeitos de transparência.
